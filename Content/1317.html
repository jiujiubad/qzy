遇到报错：前端 422 Unprocessable Entity，控制台 ajax Can't verify CSRF token authenticity


解决办法1（推荐）：app/assets/javascripts/application.js 添加 //= require jquery_ujs，虽然这个 gem 的官方 github 有说 rails5 以上版本可以不添加这行，但这里遇到问题说明还是要加上。


解决办法2（可能会有安全问题）：避开 csrf，在 controller 添加 skip_before_action :verify_authenticity_token
