课前作业（ 自习 / 线上辅导 ）

在课前准备这一个阶段，内容是免费的。在这一个阶段的准备课程，主要是有关于如何进入 Web 全栈开发的预备基础知识。内容围绕着：

  • 开发环境、编辑器建置
  • Ruby / Rails 基础知识
  • Git 基础操作 / Heroku 部署操作
  • Rails 初级项目练习

为进入实战项目学习做准备，打下扎实底子。

（注：注册为免费会员可以进行初级自习，但是付费全栈营会员享有“线上助教完全帮助”。）

初级项目实战 - 第 1 阶段（约 3 周）

全栈营课程的前3周将会以一个真实的小型项目（约 20 条需求）作为蓝本。同学将会在这个阶段学到：

  • “元学习”(学习“学习”再学习) 技能
  • 将内心的 Idea，梳理成可以执行的开发项目细项
  • 如何正确的下出 Google 关键字，并且在 Stackoverflow 找到有效答案
  • 基础项目管理
  • 基础 Rails API + 基础 CSS
  • Rails 试错技巧
  • Rails 进阶功能实践
  • 有效技术写作

带领同学体会第一次独立的“技能迁移”做项目以及“放飞”。此阶段相当于野生自学第 3-6 个月效果。此阶段结束时将有一个小型作品实做比赛。

中级项目实战 - 第 2 阶段（约 5 周）

线上全栈营课程的后 5 周将会以一个真实的中型项目（约 80 条需求）作为蓝本。同学将会在这个阶段学到：

  • 高级 User Story 编写方法
  • 多“角色”的项目实战
  • 进阶 Rails API + 进阶 CSS + 初阶 JavaScript
  • View / Helper 最佳实践
  • Controller / Model 最佳实践
  • 进阶系统架构设计
  • 第三方服务整合
  • 付费系统架构设计
  • 前端 / 后端性能调优
  • 安全性防御
  • 项目产品化技能

带领同学体会第二次独立的「技能迁移」做项目以及「线上协作」。此阶段相当于野生自学 12-18 个月效果。此阶段结束时将有一个中型作品实做比赛。将会综合两个月内学到的所有技能，「组队」进行协作项目比赛。

求职辅导 （约 1 周）

如果你顺利完成这八周的课程（必须每周验收作业），除了「结业证书」外。你还将收获毕业求职「必需」的：

  • 两个以上的真实项目经验
  • 独立技术开发能力
  • 真实产品上线能力
  • 满满的技术博客
  • 线下交流人脉
  • 远程协作技能

我们将在这个阶段指导各位如何准备履历、笔试，以及如何面试阶段的测试，帮助你为实际工作面试做好准备。
